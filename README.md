OmniROM for the Sony Xperia L
------------------


Getting Started
---------------

To get started with OMNI ROM, you'll need to get
familiar with [Git and Repo](http://source.android.com/download/using-repo).

To initialize your local repository using the OMNIROM trees, use a command like this:

    repo init -u git://github.com/taoshanROMs/manifest.git -b omni-4.4

Then to sync up:

    repo sync

Then to build:

     cd <source-dir>; . build/envsetup.sh; brunch taoshan


If you need more information or a more detailed guide, click [here to see our wiki.](http://docs.omnirom.org)
